## Github 이용 규칙
- `문풀반` 폴더와 `이론반` 폴더를 나눠, 각자 폴더를 관리합니다!
- **이론반**
  - 발제자가 발제한 자료들은 `0_발제-자료` 폴더에서 관리합니다.
  - 발제자가 선정한 문제에 대한 풀이는, `이론반` 폴더 아래 각자 개인 폴더를 만들어 해당 폴더 내부에서 문제를 풀고, Github으로 Push/PR합니다.
